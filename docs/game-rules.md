# Rules

## 百マス計算の報酬
- 今日初回: +10GP。
- 反復報酬: 正解率ポイント × スピードポイント。
- 正解率ポイント: 60%以下=0、61〜70%=1、71〜99%=2、100%=3。
- スピードポイント: 120秒以内=3、240秒以内=2、240秒超=1。

## 算数ギルド Ver1.0
- ギルドクエスト1〜5は各10問の四択。正解で次へ進み、不正解でそのクエストの1問目へ戻る。
- 再挑戦時も問題順は固定し、選択肢の位置だけを入れ替える。
- 各クエストは初回攻略のみ5GP。攻略後は再挑戦できない。
- 後続クエストは、前段階の問題を5問含めて学習内容を積み重ねる。
- ギルドテスト1はクエスト1〜5攻略後に解放。全問題群から20問をランダム出題する。
- 正解率90％以上で合格。合格5回で称号が「素人」から「見習い」になる。
- テスト報酬は正解率ポイント×スピードポイント。正解率90〜99％=1、100％=2。180秒以内=2、300秒以内=1、300秒超=0。
