# Rules

## 百マス計算の報酬
- 今日初回: +10GP。
- 反復報酬: 正解率ポイント × スピードポイント。
- 正解率ポイント: 60%以下=0、61〜70%=1、71〜99%=2、100%=3。
- スピードポイント: 120秒以内=3、240秒以内=2、240秒超=1。

## 算数ギルド Ver1.0
- ギルドクエスト1〜5は各10問の四択。正解で次へ進み、不正解でそのクエストの1問目へ戻る。
- 再挑戦時も問題順は固定し、選択肢の位置だけを入れ替える。
- 各クエストは初回攻略のみ5GP。攻略後は再挑戦できない。
- 後続クエストは、前段階の問題を5問含めて学習内容を積み重ねる。
- ギルドテスト1はクエスト1〜5攻略後に解放。全問題群から20問をランダム出題する。
- 正解率90％以上で合格。合格5回で称号が「素人」から「見習い」になる。
- テスト報酬は正解率ポイント×スピードポイント。正解率90〜99％=1、100％=2。180秒以内=2、300秒以内=1、300秒超=0。


## 算数ギルド Ver2.0 追加ルール

- ギルドテスト1に5回合格すると「見習い」となり、ギルドクエスト6が解放される。
- ギルドクエスト6〜10も各10問。正解で次へ進み、不正解でそのクエストの1問目へ戻る。
- 各クエストは初回攻略時のみ5GPを受け取り、攻略後は再挑戦できない。
- ギルドクエスト6〜10をすべて攻略するとギルドテスト2が解放される。
- ギルドテスト2はクエスト6〜10の問題からランダム20問を出題し、何度でも挑戦できる。
- テストは正解率90%以上で合格。タイムと正解率の掛け算で、控えめなGPを受け取る。
- ギルドテスト2に5回合格すると称号「徒弟」が認定される。

## 社会ギルド Ver1.0「世界のすがた」

- 対応範囲はテキスト p.26〜28。
- ギルドクエスト1〜5は各10問。クエスト2以降は、前半5問が直前クエストの復習、後半5問が新しい内容。
- 正解で次へ進み、不正解でそのクエストの1問目へ戻る。
- 各クエストは初回攻略時のみ5GP。攻略後は再挑戦できない。
- クエスト1〜5をすべて攻略するとギルドテスト1が解放される。
- ギルドテスト1は、各クエストの新規学習範囲から4問ずつ、合計20問をランダム出題する。完全ランダムではなく、必ず全範囲を網羅する。
- ギルドテスト1は何度でも挑戦できる。正解率90％以上で合格し、合格かつ5分以内で2GP。
- 「ふりかえりの修行」の社会「世界のすがた」では、社会ギルドの問題群から5問をランダム出題する。
